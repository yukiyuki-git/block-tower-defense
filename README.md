# 方块塔防 — 独立开发2D塔防游戏

使用 Godot 4.6 和 GDScript 从零构建的2D塔防游戏。

## 演示视频

https://github.com/yukiyuki-git/block-tower-defense/assets/演示视频.mp4


## 核心玩法

- **8波递增难度**的敌人进攻，每3波出现Boss敌人
- **4种防御塔**：基础塔、冰塔（减速）、火焰塔（范围伤害）、激光塔（远程高速）
- **塔升级系统**：最高3级，提升伤害、射程、攻速
- **金币经济**：击杀敌人获得金币，用于建造和升级
- **Boss系统**：体积更大、血量更高、击败获得双倍奖励
- **程序化动画**：行走弹跳、摇摆和攻击缩放动画

## 操作方式

| 操作 | 按键 |
|------|------|
| 放置塔 / 升级 | 鼠标左键 |
| 移动视角 | 鼠标右键拖动 |
| 缩放 | 滚轮 |
| 切换塔类型 | 数字键 1-4 |
| 开始游戏 | 空格 |
| 重新开始 | R |

## 技术要点

- **TileMap网格系统**：基于坐标区域检测限制塔的放置位置
- **Area2D碰撞检测**：实现攻击范围判定和敌人进入/离开事件
- **动态节点生成**：通过字符串拼接GDScript代码实现波次属性差异化
- **Canvas2D绘制**：实时绘制敌人血条和塔的攻击范围可视化
- **程序化动画**：使用sin波函数生成行走弹跳和攻击缩放动画
- **Camera2D**：实现鼠标拖拽平移和滚轮缩放的地图浏览

## 技术栈

- **引擎**：Godot 4.6
- **语言**：GDScript
- **美术**：程序化生成 + 自绘素材
